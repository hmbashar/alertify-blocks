A collection of beautiful alert/notice blocks for the WordPress block editor.

== Description ==

Alertify Blocks adds a collection of beautiful and customizable alert/notice blocks to your WordPress editor. Perfect for displaying important messages, warnings, success notifications, and more on your website.

= Features =

* Multiple alert types (Info, Success, Warning, Error)
* Dismissible notices with customizable duration
* Rich text editing support
* Customizable colors and typography
* Spacing controls
* Clean and modern design
* Lightweight and fast

= Alert Types =

* Info - For general information messages
* Success - For positive feedback and confirmations
* Warning - For important notices that need attention
* Error - For critical messages and errors

= Usage =

1. Install and activate the plugin
2. Open the block editor
3. Search for "Alertify" or "Notice" in the block inserter
4. Choose your desired alert type
5. Customize the content and styling as needed

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/alertify-blocks` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the block editor to add alert blocks to your content

== Frequently Asked Questions ==

= Can I customize the colors? =

Yes, each alert block supports custom background and text colors through the block editor's color settings.

= Do the dismissible notices persist across sessions? =

Yes, when a user dismisses a notice, it stays dismissed using local storage until the specified duration expires.

= Are the blocks responsive? =

Yes, all alert blocks are fully responsive and look great on all devices.
